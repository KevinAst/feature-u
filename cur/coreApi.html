Core API

Core API

The two primary functions you will use in feature-u are createFeature() and launchApp().

The following list summarizes the Core API. For your convenience this list:

  1. summarizes the functional API signatures
  2. includes related callback/hooks (indented)
  3. links to the detailed API docs

Core API Summary

  • createFeature({name, [enabled], [publicFace], [appWillStart], [appDidStart], [pluggableAspects]}): Feature
    • appWillStart({app, curRootAppElm}): rootAppElm || falsy
    • appDidStart({app, [appState], [dispatch]}): void
  • launchApp({[aspects], features, registerRootAppElm}): App
    • registerRootAppElm(rootAppElm): void
  • managedExpansion(managedExpansionCB): managedExpansionCB
    • managedExpansionCB(app): AspectContent
